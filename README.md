# Daily Inspirational Quote

A simple, elegant web app that displays a random inspirational quote. Perfect for a quick dose of motivation—open the page, read the quote, and tap **New Quote** for another.

## Features

- **Curated quotes** — Dozens of short, memorable quotes about success, courage, and growth
- **Lightweight** — Plain HTML, CSS, and JavaScript; no build step or framework
- **Responsive** — Works on desktop and mobile
- **Polished UI** — Gradient background, card layout, and smooth fade when changing quotes

## Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Custom properties, flexbox, gradients, transitions
- **Vanilla JavaScript** — Random quote selection and DOM updates
